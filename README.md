# Interactive3DWebApp
Interactive ThreeJS Web App — viewer minimal pour visualiser un modèle GLB avec hotspots.

Run local dev server (préféré pour éviter les problèmes CORS) :

```bash
# avec npx http-server (Node.js requis)
npx http-server . -p 8080 --cors

# ou avec python 3 (serveur simple)
python -m http.server 8080
```

Ouvrez ensuite `http://localhost:8080` dans votre navigateur.
